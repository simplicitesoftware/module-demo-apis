Apache Maven
============

Build
-----

``` text
mvn clean package
```

Javadoc
-------

``` text
mvn javadoc:javadoc
```

Checkstyle (optional)
---------------------

``` text
mvn checkstyle:check
```

Typescript
==========

Compile (optional, requires node.js)
------------------------------------

``` text
npm install
npm run tsc
```
